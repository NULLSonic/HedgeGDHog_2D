***CREDITS TO THE ORIGINAL FRAMEWORK/ENGINE THAT THIS REPO IS BASED OFF!!***

HedgeGDHog Engine (2D) is a [Sonic Worlds Next](https://github.com/Techokami/SonicWorldsNext) fork which adds new stuff some from the modern games like the stomp, Homing Attack etc. but also other stuff such as adding continues and game/time overs. A lot of changes are also made like the HUD got some tweaks to its sprites and code.

This framework is being used in these fangames:
- [Sonic | Classics Re-Imagening Project](https://gamejolt.com/games/scrproject/896662)
- More to come (maybe) :P
