## About
HedgeGDHog Engine (2D) is a [Sonic Worlds Next](https://github.com/Techokami/SonicWorldsNext) fork which adds new stuff some from the modern games like the stomp, Homing Attack etc. but also other stuff such as adding continues and game/time overs (sprites at least).
A lot of changes are also made like the HUD got some tweaks to its sprites and code.

Each playable character also get their respective Mania Sprites for some more fluid animtions.
